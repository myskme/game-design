# 手动步骤 + 给 Claude Code 的口令

这份文件把需要「人」做的事和可以「让 Claude Code 做」的事分开。
在另一台能跑 Claude Code 的 MacBook 上,按下面顺序操作。

---

## 第一部分 · 你本人先做(2 件事)

### 1. 想好两个信息

- **vault 路径**:你的 Obsidian 笔记库在那台 MacBook 上的位置,
  例如 `~/Documents/MyBrain`。
- **仓库名**:给 vault 用的私有仓库名,例如 `myskme/brain`。

### 2. 建私有 GitHub 仓库(可跳过,见口令第 1 步)

如果那台电脑装了 `gh` 命令行且已登录,Claude Code 能自己建仓库,跳过即可。
否则手动建:打开 https://github.com/new → 仓库名 `brain` →
**选 Private** → 不要勾 README → 创建。

> ⚠️ 必须 Private。笔记是私人内容。

---

## 第二部分 · 复制下面这段「口令」给 Claude Code

在那台 MacBook 上启动 `claude`,把下面整段粘贴进去。
**先把两个尖括号 `< >` 里的内容换成你自己的。**

```
我要把我的 Obsidian 笔记库接入 Git 同步和 Claude,请按步骤执行。

背景信息:
- 我的 Obsidian vault 路径:<在这里填,例如 ~/Documents/MyBrain>
- vault 用的 GitHub 私有仓库:<在这里填,例如 git@github.com:myskme/brain.git>

请执行:
1. 检查上面这个 GitHub 仓库是否已存在。如果还没建:
   - 若本机装了 gh CLI 且已登录,用 gh 创建一个 private 仓库;
   - 否则停下来告诉我,我去 github.com/new 手动建一个 Private 仓库再继续。
2. 把公开仓库 https://github.com/myskme/game-design.git 克隆到一个临时目录,
   切到分支 claude/obsidian-ai-integration-FsTz6,里面有 obsidian-second-brain/ 套件。
3. 确认我的 vault 路径存在、且是 Obsidian 库(含 .obsidian 目录)。
   如果这台电脑上没有这个 vault,停下来告诉我(说明这台电脑不是主力设备)。
4. 运行套件脚本:
   bash <临时目录>/obsidian-second-brain/scripts/setup-vault.sh <我的vault路径> <我的仓库地址>
5. 按脚本输出的提示,cd 到 vault 目录执行 git push -u origin main。
   首次推送若需要 GitHub 认证,停下来让我处理。
6. 推送成功后,报告笔记总数和收件箱待整理数量,并提醒我去 Obsidian 装插件。

硬性要求:不要删除或覆盖我已有的任何笔记;脚本对已存在的配置文件会自动跳过。
```

口令跑完后,vault 已经在 GitHub 上、并带好了 `CLAUDE.md` 和 Claude 配置。

---

## 第三部分 · 口令之后你本人做(GUI 操作,Claude 做不了)

### A. 这台 MacBook 装 Obsidian Git 插件

Obsidian → 设置 → 第三方插件 → 关闭安全模式 → 浏览社区插件 →
搜 **Obsidian Git** → 安装 + 启用。进设置改:
- Vault backup interval: `10`
- Auto pull interval: `10`
- Pull updates on startup: 开
- Push on backup: 开

### B. 第二台 MacBook

终端克隆,再用 Obsidian「打开文件夹作为 vault」,同样装 Obsidian Git 插件:
```
git clone <你的仓库地址> ~/路径/到你的vault
```

### C. iPhone / iPad

在 Obsidian app 里装 Obsidian Git 插件,用 HTTPS + GitHub Personal Access
Token 认证(token 勾 `repo` 权限),执行一次 Clone。详见 README 第 5 步。

### D.(可选)接入 Claude Code 网页端做云端自动化

打开 https://claude.com/code,授权 vault 仓库,建环境。详见 README 第 7 步。
